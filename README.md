## How to use :
- `./mandelbrot`          : Basic version in ASCII
- `./mandelbrot -color`   : Color version

- You can also use your keyboard to move around:
    - Moving `W` `A` `S` `D`
    - Zooming `Q` `E`
    - Adding/Decreasing iterations `I` `K`
    - Exit `CTRL` + `D`
